

# Features to be added

## self-build house - OK
- Add a new block or item that allows players to easily build a simple house structure
- Hold this item and right-click to place a pre-defined house structure in the world
- House can be hardcoded or generated from a template
- Starting with a simple small wooden house to ensure it works, then expand to more complex designs or different materials

## Chest organizer
- Add new button to chests GUI that, when clicked, automatically sorts the chest's contents
- Auto merge stacks of the same item and sort them in a consistent order
- Sorting can be based on item type, rarity, or custom categories

## BGM player
- Add a new feature that allows players to play custom background music in the game
- Allow players to select music files from a specific folder(Userprofile/Music, eg) and play them in-game
- Provide basic controls for play/pause/stop and volume adjustment

## Minimap
- Add a minimap feature that displays a small map of the surrounding area in the corner of the screen
- Minimap should show terrain, nearby entities, and player location
- Allow players to toggle the minimap on/off and customize its appearance (size, transparency, etc.)
- Allow players to set waypoints on the minimap for navigation purposes
